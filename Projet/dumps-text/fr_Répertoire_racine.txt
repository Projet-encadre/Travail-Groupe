   Apparence
   (BUTTON) déplacer vers la barre latérale (BUTTON) masquer
   Un article de Wikipédia, l'encyclopédie libre.
   [70][220px-FileFolders.svg.png] root est le répertoire racine de ce
   système de fichiers

   Dans les [71]systèmes de fichiers informatiques, la racine des
   répertoires ou dossiers (appelé aussi la racine d'un disque) est la
   base des répertoires dans la hiérarchie. Il peut être imagé par un
   tronc d'un arbre, c'est-à-dire le point de départ de toutes les
   branches (de tous les répertoires/dossiers).

   Pour reprendre l'analogie avec le monde réel, dans une armoire à
   classeurs (ou trieur), les tiroirs sont représentés comme le plus haut
   niveau des sous-répertoires du système de fichiers, et la pièce peut
   être considérée comme étant le répertoire racine. Le répertoire racine
   peut contenir d'autres répertoires mais il ne peut être contenu, du
   moins dans ce système de fichiers.

   Note : dans les [72]systèmes d'exploitation populaires comme
   [73]Microsoft Windows et [74]Unix, il est possible de placer des
   [75]fichiers dans la racine du disque.

   Sous Unix, la racine des répertoires est noté /. Toutes les entrées, y
   compris les partitions montées, sont placées de façon hiérarchique,
   comme les branches d'un arbre, par rapport à la partition racine.
   Cependant, sous [76]DOS et Windows, chaque partition (ou lecteur)
   possède une racine (C:\ pour la partition principale [77]par défaut
   sous Windows).
